# 🚀 Student Hub (JEET)

A modern **Productivity Dashboard** built using **React + Vite + Tailwind CSS**, featuring a powerful **Spotlight Search (Ctrl + K)** for fast navigation between tools.

---

## ✨ Features

* 🔍 **Spotlight Search (Ctrl + K)**
* ⚡ Instant navigation between tools
* 🎯 Keyboard controls (↑ ↓ Enter Esc)
* 🌙 Dark / Light mode
* 🧊 Glassmorphism UI
* 🎞️ Smooth animations (Framer Motion)

---

## 🧰 Tools Included

* ⏰ Clock
* ⏱️ Stopwatch
* 🍅 Pomodoro Timer
* 📅 Calendar
* ✅ Todo List
* 🔔 Alarm
* 🧮 Calculator
* 📊 Analytics (Dashboard cards)

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 Tailwind CSS
* 🎞️ Framer Motion
* 🔥 Font Awesome
* 🌐 React Router

## ⌨️ Keyboard Shortcuts

| Key      | Action             |
| -------- | ------------------ |
| Ctrl + K | Open Spotlight     |
| ↑ / ↓    | Navigate results   |
| Enter    | Open selected tool |
| Esc      | Close Spotlight    |
